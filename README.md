# META‑CAR‑SHOWROOM

## Overview
A high‑fidelity Unity HDRP (High Definition Render Pipeline) showcase of a luxury car. The project demonstrates realistic lighting, materials, and post‑processing effects, providing a solid foundation for developers to build upon or integrate into larger applications.

## Features
- **HDRP Rendering** with physically‑based materials
- Dynamic lighting and skybox
- Real‑time reflections and post‑processing effects
- Fully animated car model with controllable camera
- Optimized for macOS and Windows builds

## Requirements
- Unity **2021.3 LTS** or newer (HDRP package installed)
- macOS 10.15+ or Windows 10+ (for building)

## Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/pavanganeshpg/META-CAR-SHOWROOM.git
   cd META-CAR-SHOWROOM
   ```
2. **Open the project in Unity**
   - Double‑click the `META-CAR-SHOWROOM` folder or run the helper script:
     ```bash
     ./setup.sh
     ```
   - Unity will open the project and load the `MainScene`.
3. **Play the scene**
   - Press the Play button in the Unity editor to explore the showroom.

## Building the Project
- Open **File → Build Settings** in Unity.
- Add the `MainScene` to the build.
- Choose your target platform (macOS, Windows) and click **Build**.
